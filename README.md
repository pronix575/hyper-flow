# Hyper Flow
## Framework for building progressive console applications on node.js platform
## setup ⚙️
```bash
cd hyper-flow

yarn setup
# or
npm setup
```
## build 🛠
```bash
yarn build
```
## start 🚀
```bash
yarn start
```

## developing 🧱
```bash
yarn dev:build
```
```bash
yarn dev:start
```
